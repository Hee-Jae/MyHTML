HTML5 도입 후 새로운 기능

HTML5의 새로운 폼

< input type="number" name=" " min="a1" max="a2" > : 최소 a1부터 최대 a2 까지의 숫자만 입력 가능하다.
최소10, 최대 20
< input type="color" name=" " > : 색상 선택이 가능하다.

< input type="date" name=" " > : 날짜 선택이 가능하다.

< input type="email" name=" " > : 이메일 입력이 가능하다. 이메일 형식이 아니면 데이터전송을 거부한다.

< input type="month" name=" " > : 날짜를 '달'까지 고를 수 있다.

< input type="range" name=" " min="a1", max="a2"> : a1부터 a2까지 숫자를 고를 수 있다. 선택 바가 상당히 애매하다.

< input type="search" name=" " > : 검색창이다. text랑 별다를게 없어 보인다.

< input type="tel" name=" " > : 전화번호 입력이 가능하다.

< input type="time" name=" " > : 시간 입력이 가능하다.

< input type="url" name=" " > : url주소 입력이 가능하다.

< input type="week" name=" " > : 한 해의 몇번 째 주 선택이 가능하다.


HTML5의 새로운 속성

< form autocomplete="on" > < /form >: 사용자의 의사에 따라 자동완성
< input type="text" autocomplete="off" > : 이런식으로 특정 입력 칸은 자동완성기능을 없앨 수 있다.
< input type="text" placeholder="내용" > : 입력하기 전에 내용을 띄우며 내용 입력시 사라진다.

< input type="text" autofocus > : 페이지 접속 시 자동으로 커서가 생긴다.

< input type="text" required > : 이 칸은 반드시 입력을 해야 데이터전송이 가능하다.

< input type="text" pattern="정규표현식" > : 입력값을 표현식에 따라 제한한다.

정규표현식

[a-z] : a부터 z까지 한 글자
[A-Z] : A부터 Z까지 한 글자
[0-9] : 0부터 9까지 한 숫자
. : 모든 문자
+ : 문자 한개 이상
예시 : [a-zA-Z].+[a-zA-Z]
첫 자리에는 a~z 또는 A~Z 가 있어야 하며, 그 이후엔 모든 문자가 여러자리 올 수 있고, 마지막 자리에는 a~z 또는 A~Z 가 있어야 한다.
