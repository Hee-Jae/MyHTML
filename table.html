표 만들기

실제 데이터를 위한 태그

< table > < /table > : 표 생성
< tr > < /tr > : table row 표의 행
< td > < /td > : table data 표의 데이터
< td colspan="n" > : n 개의 열 병합. < td rowspan="n" > : n 개의 행 병합.

명시적인 표현을 위한 태그

< thead > < /thead > : 표의 제일 머리 부분
< tbody > < /tbody > : 표의 내용 부분
< tfoot > < /tfoot > : 표의 제일 밑부분

표의 속성을 위한 태그

< table border="숫자" > : 표의 테두리 굵기
< th > < /th > : 표의 머리 부분의 데이터. 글씨가 굵어진다.

표 만들기 예시

< table border="1">
< thead >
< tr >< th >제목1< /th >< th >제목2< /th >< th >제목3< /th >< /tr >
< /thead >
< tbody >
< tr >< td >내용1< /td >< td >내용2< /td >< td >내용3< /td >< /tr >
< tr >< td >내용4< /td >< td >내용5< /td >< td >내용6< /td >< /tr >
< /tbody >
< tfoot >
< tr >< td >꼬리1< /td >< td >꼬리2< /td >< td >꼬리3< /td >< /tr >
< /tfoot >
< /table >
