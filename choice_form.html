선택폼 만들기

< select name=" "> < /select > : 선택이 가능한 폼 만들기. name속성은 이 값을 구분해주는 이름.
< option value=" "> < /option > : 선택 리스트 추가하기. value속성은 위의 name과 동일한 역할.
< select name=" " multiple > : 다중 선택이 가능한 선택폼.

체크박스 만들기

< input type="radio" name=" " value=" " > : 동그라미 체크 박스. name속성은 이제 설명 안해도 다 알겠지? value 역시 option 태그의 value와 같은 것. 이 radio 타입의 경우 name이 다른 것과 일치하면 중복선택이 불가능하다.

동그라미 체크 박스 예시
빨간색 파란색 초록색

< input type="checkbox" name=" " value=" " > : 네모난 체크 박스. 속성의 경우 radio와 똑같고, 다른점은 name이 일치해도 중복선택이 가능하다.
< input type="checkbox" name=" " value=" " checked > : 미리체크
