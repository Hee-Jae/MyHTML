HTML의 기본구조

< !DOCTYPE HTML >
< html >
< head >
웹페이지 기본 설정. meta 태그, title 태그 등등
< /head >
< body >
웹페이지의 내용이 들어가는 부분.
< /body >
< /html >


-------------------------------------------------------------------
    < !DOCTYPE HTML >
