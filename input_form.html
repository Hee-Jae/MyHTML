입력폼 만들기

FORM에 대해서

< form > < /form > : 하나의 폼 생성. 입력한 정보들을 처리하는 단위가 된다.
< form action = "서버주소"> : 입력된 정보들을 해당 서버로 보낸다.

< input type="text" name="id"> : 문자를 입력할 수 있는 칸 생성. name속성은 서버에 전송할 때 이 값을 구분하는 기준이 되는 이름이다.


< input type="password" name="pwd"> : 비밀번호를 입력할 수 있는 칸 생성. name속성의 의미는 위와 동일하다.


< input type="text", value="미리쓰기"> : 텍스트 칸을 미리 채워놓는다.


< textarea cols="n" rows="m" > < /textarea> : n개의 문자를 m줄 입력할 수있는 칸 생성.


< inputy type="hidden" value="입력 값" > : 보이지 않는 텍스트 칸. 보이지 않기에 따로 입력할 수 없고 정해진 값만이 입력된다. 서버에 전송할 때 이 값 역시 함께 전송된다.
[보이지 않는 예시. 있긴 있다.]

< input type="file" name="uploadfile" > : 파일 업로드가 가능한 상자. 이 경우 필수적으로 폼 형태를 다음과 같이 지정해야 한다.
< form action="주소" method="post" enctype="multipart/form-data" >


< input type="reset" > : 사용자가 입력한 정보를 모두 지운다.


< input type="submit" value="제출버튼 이름" > : 입력한 정보들을 다른 서버로 보낸다.
