글씨체와 문단구조

1. < h1 > < /h1 > : 제목에 해당하는 부분. 글씨체를 굵고 크게 만든다.

< h1 >의 예시

2. < h2 > < /h2 > : < h1 > 의 하위 버전 (차이가 거의 없다.)

< h2 >의 예시

3. < h3 > < /h3 > : < h2 > 의 하위 버전 (얘부터 차이가 난다.)

< h3 >의 예시

4. < strong > < /strong > : 글씨를 굵게 만든다.
< strong > 의 예시

5. < br > : 줄 띄우기. 예시는 줄을 띄움으로서 생략한다.

6. < p > < /p > : 해당 부분을 하나의 문단으로 처리한다.

< p > 의 예시이다. 예시를 보여주기 위해 좀 글을 길게 써보겠다. 사실 쓸말은 없지만 정말 예시를 보여주기 위함이다. 이정도면 충분히 긴가? 혹시 모르니 한마디만 더 적겠다. 진짜 마지막으로 한마디만 더 적겠다. 아.. 살짝 부족할 거 같은데 ㅇㅋ 추해보이니까 여기까지
