이미지 만들기

< img src="위치\이미지 파일이름" > : 이미지 생성
< img src="파일이름" width="너비" height="높이"> : 이미지의 너비와 높이 조정
< img src="파일이름" alt="사진내용" > : 사진이 깨졌을 때 대체문구가 나옴

너비와 높이 둘중 하나가 생략되어 있을 경우 사진의 원래크기에 비례하여 다른 한쪽은 자동 조정된다.
둘다 생략되어 있을 경우 사진의 원래크기가 나온다.
