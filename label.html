레이블 달기

< label > < /label > : 이것은 태그에 이름표를 다는 것? 여러 입력상자를 만들었는데 각각의 입력상자가 어떤 데이터를 의미하는지 모른다면 혼란스러울 것이다. 예를들어 아이디를 입력하는 상자가 있는데 이것이 아이디를 입력하는 곳인지 이름을 입력하는 곳인지 유저들은 모른단 말이다. 그래서 이름표를 달아주는 거다. 솔직히 무슨 소리인지 잘 모르겠다. 그냥 지어낸 말이다.

사용방법

< label for="~"> 아이디 < /label >
< input type="text" id="~">
label의 for와 input의 id가 일치하면 "아이디"를 마우스로 클릭 했을 때 해당하는 텍스트상자에 커서가 생긴다.


더 쉬운방법

< label > 아이디 < input type="text"> < /label > : 이렇게하면 id와 for가 없어도 된다.
