폼의 메소드

< form action="주소" method="get" > : 서버에 어떤 데이터를 보낼 때 url에 그 데이터가 뜬다. url을 이용해 데이터를 전송하는 방법. method 속성을 생략하면 기본적으로 get으로 설정된다.

< form action="주소" method="post" > : url이 아닌 다른 방식으로 데이터를 전송한다. url에 데이터가 뜨지 않는다.

form의 예시(실제로는 없는 주소에 보내 보겠다)

< form action="http://localhost/form.php" method="get" >
아이디:< input type="text" name="id" >
비밀번호:< input type="password" name="pwd" >
< input type="submit" value="로그인" >
< /form >

아이디:
비밀번호:

아이디와 비밀번호를 입력하고 로그인을 눌렀을 때 들어가지는 페이지의 url을 보면 입력한 정보들이 나와있다.

< form action="http://localhost/form.php" method="post" >
아이디:< input type="text" name="id" >
비밀번호:< input type="password" name="pwd" >
< input type="submit" value="로그인" >
< /form >

아이디:
비밀번호:

아이디와 비밀번호를 입력하고 로그인을 눌렀을 때 들어가지는 페이지의 url을 보면 입력한 정보들이 보이지 않는다.
